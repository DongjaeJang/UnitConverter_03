### 2.1 잔여 과제 (P1 전, 1줄)

`UnitRegistry`로 단위·비율 **등록**은 분리했으나, `convert_all()` 안에는 `meter` / `feet` / `yard` **단위별 분기**가 아직 남아 있다 — P1 `register()` 도입 시 registry 기반 일반 변환 루프로 제거 예정.
